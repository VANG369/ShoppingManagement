                = Write data

      |--> order_service.py
            -> Import all services systems (Prodcut, Order)
            -> Create file CSV following systems(Product, Order)
            -> Create class with func as operation system follow rules below:
                > Load_Product
                    - Create def laod_products()
                      $ Create empty list of product. This list will store all product after return value in loop
                      $ Read data from the file
                      $ Write all data via loop as row follow index with header file and use try & except to check condition
                > Add_Product
                    - Create def add_product
                      $ Input all information with check condition values
                      $ Append_Data with (file_name, header, list)
                > Delete_Product
                    - Create def Delete_Product()
                      $ Assign products = self.laod_products
                      $ Check condition if products == 0
                      $ Use loop show the product available
                      $ Input product_id
                      $ Assign selected_product = none
                      $ Use conditon in loop to check about the product_id if equle 
                      $ Use condition to check selected_product if none...
                      $ Use var confirm assign input selected_product.name as yes/no
                      $ Check condition if comfirm "yes": products.remove(selected_product)
                      $ Use file_handler as write data...
                > View_Stock
                    - Create def view_stock
                      $ Get product via varible from load_product
                      $ Check condition product if empty product
                      $ Show products if available in stock
                > Order_Product
                    - Create def order_product
                      $ Get product via varible from load_product
                      $ Check condition product if empty product
                      $ Show products if available in stock
                      $ Input information(customer_name, product_id, quantity)
                      $ Use try & except : quantity as integer
                      $ Check condition if quantity <= 0  
                      $ Find selected product
                      $ Find product via loop in loop via product_id & product
                      $ Check product in none and show available product
                      $ Check condition if quantity > product in stock
                      $ Calculator total_price
                      $ Generate order ID 
                      $ Creat order object
                      $ Save order as file 
                      $ Update stock with selected_product.stock -= quantity
                      $ Use loop to append all product in file product_file 
                > Invoice_Order
                    - Create def invoice_order
                      $ Read data 
                      $ Check codition if no order
